# Data Folder

This folder contains the data files used for the Steam playtime prediction project.

## Included in this repository

- `steam_games.json`  
  Smaller Steam game metadata file used by the notebook.

## Not included in this repository

- `steam_new.json`  
  This file is too large to store on GitHub, so it is intentionally excluded from version control.

To run the project, place `steam_new.json` inside this `data/` folder locally:

```text
data/steam_new.json
